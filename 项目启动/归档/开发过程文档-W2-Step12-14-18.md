## Step 12：TF 卡同步引擎

### 模块架构

```
src-tauri/src/sync/
├── mod.rs              # 模块导出
├── detector.rs         # TF 卡/可移动卷检测（扫描 /Volumes）
├── manifest.rs         # .arca/manifest.json 解析
├── session_parser.rs   # 会话目录解析（audio/photos/scans/live_clips）
├── file_copier.rs      # 文件复制（增量 + 大小校验）
├── meta_parser.rs      # .meta.json AI 元数据解析
├── timeline_builder.rs # 时间轴自动构建（Timeline + AudioTrack + Keyframe）
├── state.rs            # sync_state.json 同步状态管理（增量同步）
└── progress.rs         # Tauri Event 进度推送

src-tauri/src/commands/sync.rs  # IPC 命令：scan_tf_card, preview_import, import_sessions, get_sync_status
```

### 同步流程

```
扫描 /Volumes → 识别 .arca 目录 → 解析 manifest.json
  → 预览可导入会话（增量过滤）
  → 用户确认 → 创建 Project → 复制文件 → 解析元数据
  → 写入 Asset + AI 分析 → 构建时间轴 → 更新同步状态
```

### 新增 Cargo 依赖

| 包 | 用途 |
|----|------|
| dirs-next | 获取系统数据目录 |
| walkdir | 递归遍历会话目录 |
| notify | 文件系统事件监听（预备） |

### IPC 命令

| 命令 | 参数 | 返回 |
|------|------|------|
| `scan_tf_card` | 无 | `{ cards: DetectedCard[] }` |
| `preview_import` | `arcaPath` | `ImportPreview` |
| `import_sessions` | `arcaPath, sessionIds, libraryId` | `string[]`（projectIds） |
| `get_sync_status` | `arcaPath` | `SyncedSessionRecord[]` |

## Step 13：音频处理核心

### 模块架构

```
src-tauri/src/audio/
├── mod.rs          # 模块导出
├── decoder.rs      # Symphonia 音频解码器（m4a/wav/mp3/aac → PCM f32 单声道）
├── waveform.rs     # 波形数据生成（100 peaks/sec）+ 缓存机制
└── metadata.rs     # 音频元信息提取（不完整解码）

src-tauri/src/commands/audio.rs  # IPC 命令：get_audio_metadata, get_waveform_data
```

### 技术要点

| 项目 | 实现 |
|------|------|
| 解码库 | Symphonia（纯 Rust，支持 mp3/aac/isomp4/wav/pcm） |
| 多声道处理 | 自动 downmix 到单声道（取均值） |
| 波形精度 | 100 peaks/second（min + max 双值） |
| 缓存策略 | 基于文件路径 hash → `waveform_{hash}.json` |
| 缓存目录 | `AppCacheDir/waveforms/` |

### IPC 命令

| 命令 | 参数 | 返回 |
|------|------|------|
| `get_audio_metadata` | `filePath` | `AudioMetadata` |
| `get_waveform_data` | `filePath` | `WaveformData` |

## Step 14：前端波形渲染引擎

### 组件架构

```
src/components/features/timeline/
├── TimelineView.tsx        # 时间轴整体容器（组合所有子组件）
├── WaveformRenderer.tsx    # Canvas 波形渲染（双色：海军蓝 + 金色已播放）
├── Playhead.tsx            # 播放头（2px 金色竖线 + 8px 圆形指示器）
├── TimeRuler.tsx           # 时间刻度尺（自适应密度）
├── PlaybackControls.tsx    # 播放控制栏（⏮ ◀◀ ▶/⏸ ▶▶ ⏭ + 速度 + 音量）
└── SelectionOverlay.tsx    # 时间范围选区（Alt+拖拽）

src/hooks/
├── useAudioPlayer.ts       # Web Audio API 封装（播放/暂停/跳转/速度/音量）
├── useTimelineDrag.ts      # 波形水平拖拽滚动
├── useTimelineZoom.ts      # 波形缩放（Ctrl/Cmd + 滚轮）
└── useTimelineShortcuts.ts # 键盘快捷键（Space/←→/Home/End/Ctrl±0）
```

### 视觉规格实现

| 属性 | 值 |
|------|------|
| 未播放波形色 | `rgba(31, 69, 110, 0.6)` |
| 已播放波形色 | 渐变 `#FFD54F → #FFC000` |
| 播放头 | 2px 宽 `#FFC000` + 顶部 8px 圆形 |
| 选区高亮 | `rgba(31, 69, 110, 0.10)` |
| 时间刻度 | 自适应间距（80px 目标） |

### 快捷键

| 键 | 功能 |
|----|------|
| Space | 播放/暂停 |
| ← / → | 后退/前进 5 秒 |
| Home / End | 跳转开头/结尾 |
| Ctrl/Cmd + +/- | 缩放 |
| Ctrl/Cmd + 0 | 重置缩放 |

## Step 18：全局悬浮窗

### Rust 端

```
src-tauri/src/commands/dropzone.rs
  ├── create_dropzone_window  # 创建系统浮动面板窗口
  ├── close_dropzone_window   # 关闭悬浮窗
  └── toggle_dropzone_window  # 显示/隐藏切换
```

### 前端架构

```
src/stores/dropzoneStore.ts   # 状态机：Hidden → Idle → Attract → Processing → Complete

src/components/features/dropzone/
├── DropzoneWindow.tsx     # 容器（拖拽事件处理 + 状态分发）
├── DropzoneIdle.tsx       # 待机态（NC Logo，L3 材质，可点击展开）
├── DropzoneProcessing.tsx # 处理态（旋转动画 + 进度百分比）
├── DropzoneComplete.tsx   # 完成态（绿色 ✓ 动效）
└── DropzoneExpanded.tsx   # 展开态（最近 5 条导入记录列表）
```

### 状态机

```
Hidden ──(show)──▶ Idle ──(dragEnter)──▶ Attract
  ▲                  │                      │
  │                  │(click)         (drop)│
  │                  ▼                      ▼
  │              Expanded              Processing
  │                                        │
  │                          (complete)    │
  └───────(hide)───────────── Complete ◀──┘
                               │(2s timeout)
                               ▼
                              Idle
```
