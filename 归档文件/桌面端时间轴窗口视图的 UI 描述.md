NCdesktop 软件 PRD UI 界面描述
一、应用窗口概览
应用名称: NCdesktop

界面风格: Liquid Glass (流体玻璃) 设计风格。采用毛玻璃、半透明材质和柔和的光晕效果。

背景颜色: #1F456E 品牌海军蓝作为主色调，具有深色模式的沉稳感。

高亮颜色: #FFC000 品牌金色用于关键交互指示和高亮。

布局: 标准的三栏式布局：左侧导航栏、中央主内容区、右侧侧边栏 (Inspector)。

二、左侧导航栏 (KNOWLEDGE LIBRARY)
顶部: 品牌名称 "NCdesktop"，标题 "KNOWLEDGE LIBRARY"，下方有 "RCDTF" 字样（可能是设备或库的名称缩写）。

导航树 (金色图标和文字):

Search: 带搜索图标。

Recent: 带“最近”图标，当前处于高亮选中状态，具有金色背景和边框。

Projects: 带文件夹图标。展开状态下，高亮项目项目名为“Quantum Mechanics Lecture 07”，下面有“Timeline 1”子项。

Tags 树: 下面有三个品牌金色高亮标签：#Physics, #Quantum, #Schrodinger。

底部状态栏:

Settings: 设置图标和金色文字。

TF Card Connected: TF卡图标和金色文字，指示硬件已连接。

3.1 页面标题: 显示项目名称 "Quantum Mechanics Lecture 07 - March 20, 2026"。

3.4 时空记忆轴 (RECORDING AXIS / CONTEXTUAL TIMELINE):

顶部标题: "Recording Axis"

波形轨道: 深海军蓝背景，带金色波形数据。

播放控制: 左侧有金色播放/暂停图标。

播放头 (Gold line): 一个品牌金色的竖线，顶部有一个圆形 Magic Moment 指示器。

时间刻度: 播放头上方显示当前时间 09:14 (金色)。右侧显示总时长 09:14 (金色)。
