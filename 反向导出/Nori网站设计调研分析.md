# HeyNori 网站设计深度调研分析 ("暖科技"风格)

这是对 [heynori.com](https://heynori.com/) 的设计语言、布局结构及交互细节的深度拆解，作为 NCdesktop 官网设计的核心参考。

---

## 1. 视觉风格：暖科技 (Warm Tech)
Nori 避开了传统的冷色调科技感，采用了更有亲和力、更接近家居氛围的视觉体系。

### 核心要素：
| 元素 | 规格 | 目的 |
| :--- | :--- | :--- |
| **主色调** | 奶油色 (`#FDF8EE`) & 桃色 (`#FFBD74`) | 传达舒适、信任和“家”的感觉，区别于传统蓝白配色。 |
| **字体** | 圆体无衬线字 (Rounded Sans-Serif) | 软化界面，显得更友好、更安全。 |
| **图片风格** | “黄金时刻” 生活化照片 | 展示产品在真实环境（如咖啡桌、办公桌）中的使用。 |
| **形状语言** | 极大圆脚 (24px+) | 极度圆润的按钮和卡片，消除尖锐感。 |

---

## 2. 页面布局：叙事型结构 (Storytelling)
页面引导用户从好奇逐步转向行动：

1.  **英雄区 (Hero)**：
    *   **标题**：利益优先（如“让家庭生活更轻松”）。
    *   **展示**：产品在平板和手机上的多端联动效果图。
2.  **功能网格 (Feature Grid)**：
    *   采用马赛克风格，结合对话气泡和实景图。
    *   **效果**：直观展示“怎么用”，替代枯燥的说明文字。
3.  **交互式功能标签 (Interactive Tabs)**：
    *   点击切换“日历”、“餐饮”、“任务”。
    *   **效果**：展示深度的同时也保持了页面的简洁。
4.  **社交证明 (Social Proof)**：
    *   基于卡片的极简评价轮播。

---

## 3. 内容策略与交互细节
*   **文案风格**：强调 **“获益”** 而非 **“功能”**（例如：“AI 自动组织，你无需动手”）。
*   **微动效**：
    *   **Hover Lifts**：卡片悬浮时有轻微的向上漂浮感。
    *   **顺滑切换**：标签页切换过程带有柔和的淡入效果。
*   **吉祥物交互**：小猫吉祥物作为品牌记忆点，常驻在“开始使用”按钮旁。

---

## 4. 对 NCdesktop 官网的借鉴方案
1.  **场景化背景**：将 NCdesktop 置于真实的开发者、创作者办公环境中。
2.  **非白背景色**：尝试浅奶油色或浅灰色作为主基调。
3.  **Prompt 浮窗**：利用对话气泡形式展示 AI 指令，体现产品核心交互。
4.  **全面圆角**：保持 UI 的现代感和亲和力。
